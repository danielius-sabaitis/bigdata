# Big Data 2026, 1st assignment: Maritime Shadow Fleet Detection

This project processes large AIS vessel datasets to detect anomalies in vessel movement and attributes. The pipeline reads large CSV datasets in chunks, distributes them across workers, and applies anomaly detection logic to identify suspicious vessel data. 

Dataset: Danish Maritime Authority AIS data (2025-04-22, 2025-04-23)
